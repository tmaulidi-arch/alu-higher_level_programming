# Python - Hello, World

This directory contains Python scripts for the Hello, World project.

## Files

### 0-run
Shell script that runs a Python script from environment variable $PYFILE

### 1-run_inline
Shell script that runs Python code from environment variable $PYCODE

### 2-print.py
Python script that prints exactly "Programming is like building a multilingual puzzle

### 3-print_number.py
Python script that prints a number followed by "Battery street" using f-strings

### 4-print_float.py
Python script that prints a float with 2 digits using f-strings

### 5-print_string.py
Python script that prints a string 3 times and then the first 9 characters

### 6-concat.py
Python script that concatenates strings

### 7-edges.py
Python script that prints parts of a string

### 8-concat_edges.py
Python script that prints "object-oriented programming with Python"

### 9-easter_egg.py
Python script that prints "The Zen of Python"
